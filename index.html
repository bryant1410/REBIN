What is REBIN?

REBIN stands for RESTful binary wrapper and configuration dashboard - Builds RESTful endpoints for parameterized binary/script and command input and output

Creating a web service really should be this easy. Doesn't matter if you program in C, C++, Python, Ruby or even C# Mono. Expose the functionality of your app or script in just a few seconds with an API endpoint. Along with Authentication of a variety of strategies. (More are being added)

Roadmap:

  • Authentication strategy additions
  • Centralized management
  • Synchronization between REBIN instances (Scaling)
  • Load balancing controller
